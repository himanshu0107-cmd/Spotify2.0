## 🚀 About the Project

A fully functional **Spotify Clone** built with vanilla HTML, CSS, and JavaScript. It replicates the core Spotify UI and experience with a complete Indian music library.

### ✨ Features

- 🎵 **40 Indian Songs** — Bollywood, Punjabi, Indie, Retro, South Indian, Lofi, Dance & Devotional
- 🎧 **Web Audio API** — Every song plays a real synthesized musical tone (no MP3 files needed)
- 📋 **12 Indian Playlists** — Arijit Singh Hits, Bollywood Blockbusters, Punjabi Hits, Desi Party Anthems, 90s Retro, South Indian Hits, Indie Hindi Chill, Lofi Study Beats, Bhakti & Devotional, Subah Ki Shuruwaat, Highway Drive Mix
- 🔍 **Live Search** — Search by song title, artist, or genre with instant results
- 🖼️ **Real Cover Art** — Every song, album, and playlist has its own cover image
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- ▶️ **Full Player Bar** — Play, pause, next, previous, shuffle, repeat, volume control, progress scrubbing
- 🎨 **Playlist View** — Hero art, Play All button, full song list per playlist
- 📚 **Library Sidebar** — Quick access to all playlists with cover art
